# Patch Notes

Reads Path of Exile patch notes from the official forum and renders them cleanly.

Built with [Cloudflare Workers](https://workers.cloudflare.com/) + [Hono](https://hono.dev/) + [Vite](https://vite.dev/).

## Development

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```

Deploys to Cloudflare Workers at `https://patch-notes.poeshonya3.workers.dev`.
